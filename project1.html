Fire Detection Using Deep Learning

Introduction

The goal of the project is to use image processing technology to detect fires and notify people. Numerous automatic fire alarm systems already exist, such as the sensor-based method, which has limitations. This project is implemented by connecting a webcam as hardware and using the PyCharm IDE.

System Overview

The project mainly focuses on computer vision, machine learning, and image processing. We utilize Python's OpenCV library for image processing, and the webcam captures the surrounding video feed to analyze it in real-time. The project aims to detect fire as early as possible and work in various environments.

Existing System

In the existing system, fires in forests are detected using physical sensors, which can be limited by environmental factors. These systems are often unable to detect fires in the early stages, making them less effective in preventing large-scale damage. The Internet of Things (IoT) is commonly used to connect devices and exchange data in real time.
